# CompatDiffing

A set of files from ["Swift" project](https://github.com/apple/swift/tree/0d911a56c7ac7b475356b13ddc7f9e2f873fff81/stdlib/public/core)
to provide source-level compatible "diffing" features to pre-10.15 macOS.
